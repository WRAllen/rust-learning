# 前言

现在我们知道了怎么定义一个变量了，那接下来就是学习函数要怎么定义和使用。

函数就是把一段逻辑封装起来，给它起个名字，需要的时候调用一下。这点和 Python、Go 都一样，没有函数的话代码很快就会变成一坨面条。

# 函数

## 取名规范

和变量一样，Rust 的函数名也统一用 **snake_case**（小写 + 下划线），这点和 Python 一致，而 Go 偏向 camelCase。

```rust
fn calc_total_price() {}
fn get_user_by_id() {}
```

## 基本定义

Rust 用 `fn` 关键字定义函数，这点和 Go 的 `func` 有点像：

```rust
fn add(a: i32, b: i32) -> i32 {
    a + b
}
```

和 Python、Go 对比着看：

| 语言   | 写法示例                              | 特点                         |
| ---- | --------------------------------- | -------------------------- |
| Python | `def add(a, b): return a + b`    | 参数和返回值一般不用写类型             |
| Go     | `func add(a, b int) int { ... }`  | 参数要类型，返回值也要类型              |
| Rust   | `fn add(a: i32, b: i32) -> i32`   | 参数要类型，返回值用 `->` 声明，和 Go 很像 |

初学者可以先记住：**Rust 函数的参数类型、返回类型都要写清楚**，不能像 Python 那样省略。

## 调用函数

调用方式和大多数语言一样，直接写函数名加括号：

```rust
let result = add(1, 2);
println!("result: {}", result);
```

## 返回值

Rust 函数返回值有两种常见写法。

### 1. 最后一行是表达式，自动返回（推荐）

```rust
fn add(a: i32, b: i32) -> i32 {
    a + b
}
```

注意最后一行 **没有分号**。在 Rust 里，没有分号结尾的是**表达式**，表达式的值就是函数的返回值。

### 2. 用 `return` 提前返回

```rust
fn is_positive(n: i32) -> bool {
    if n > 0 {
        return true;
    }
    false
}
```

`return` 和 Go、Python 一样，用来提前结束函数并返回值。

## 表达式 vs 语句（和 03 里提到的 `;` 有关）

03 里我们说过，Rust 里大多数情况要加分号，但表达式是个例外。这里再具体说一下：

- **语句（statement）**：执行一个操作，没有返回值，一般要加 `;`
- **表达式（expression）**：会产生一个值，可以作为返回值

```rust
fn demo() -> i32 {
    let x = 3;   // 语句，有分号
    x + 1        // 表达式，没有分号，这就是返回值
}
```

如果你不小心写成 `x + 1;`，那这一行就变成了语句，函数就没有合法返回值了，编译器会报错。

说实话，这点对初学者来说有点绕，但只要记住：**函数最后一行想自动返回，就别加分号**。

## 无返回值

如果函数什么都不返回，可以省略 `->`，或者显式写成 `-> ()`：

```rust
fn say_hello() {
    println!("hello");
}
```

`()` 在 Rust 里叫 **unit 类型**，可以理解成"没有有意义的返回值"，有点像 Go 里不写返回值、Python 里隐式返回 `None` 那种感觉。

## `main` 也是函数

我们前面一直在用的 `main` 其实就是程序的入口函数：

```rust
fn main() {
    println!("Hello, world!");
}
```

只不过 `main` 比较特殊，是 Rust 程序启动时自动调用的，不需要我们手动调用。

## 函数放在哪

和 Go 不同，Rust 的函数**不能嵌套定义**（不能在 `main` 里面再写一个 `fn`）。自定义函数要写在 `main` 外面，和 `main` 平级：

```rust
fn greet(name: &str) {
    println!("hello, {}", name);
}

fn main() {
    greet("Allen");
}
```

这点和 Go 把函数写在 package 里、Python 把函数写在模块顶层是类似的。

# 小结

对于初学者，函数这块先记住这些就够了：

1. 用 `fn` 定义，函数名用 snake_case
2. 参数类型、返回类型都要写
3. 最后一行不写分号，可以自动返回
4. 想提前结束就用 `return`
5. 自定义函数写在 `main` 外面
